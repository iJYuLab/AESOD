This paper proposes a method that fully exploits shallow feature map information, providing a new research perspective for improving small object detection performance. Recently, for a specific detection task, we have further conducted a series of experiments, carried out targeted optimizations on the C2f module in the P1 stage, and appropriately introduced convolution modules with a stride of 1 into the shallow feature maps. Experimental results show that these improvements have all effectively enhanced the model’s detection accuracy.
